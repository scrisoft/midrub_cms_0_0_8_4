About Midrub Wchatbot

Midrub Wchatbot is an app for Midrub. You can't use it without Midrub.

The goal of this app is to help you to save potential clients and save people to find easy a response to their questions.

Midrub Wchatbot is an app for Midrub which allows you to create a list with Quick Replies for your Whatsapp's numbers. Each reply has a list with keywords as condition for Quick Reply and you can select even the accuracy for the keywords available in the messages.

You can connect your Whatsapp's numbers from Twilio and select one or several categories. Then, you have to select the same category or categories(at least one should be), and the bot will select the Quick Replies based on the number's categories which are used even in the Quick Replies. The bot will search for the best match and will reply. When the Midrub Wchatbot replies to a message, it saves the user as subscriber with the reply's categories and saves the conversation. You can see the subscribers and stats in both Quick Reply's page and in the History's page.

If no Quick Replies available based on the message content, you can create a default message which will be sent to your contacts if no Quick Reply available.

The Midrub Wchatbot app is able to identify the phone numbers and email addresses in the Whatsapp's messages. All of them will be saved in the Phone Numbers or Email Addresses pages. You can anytime see the message from where was extracted the phone number or email address. All emails and phone numbers can be exported in a CSV file.

The Subscribers page shows the subscribers saved by the bot and you can send to them promotional messages from the Midrub Wmarketing app.

The Audit Logs page shows in real time the bot activity. You will see the number of the replies per day and by selected Whatsapp's number. In addition, you will see the most popular Quick Replies by selected Whatsapp's number.

Bot replies could be limit based on the plans. The Midrub Wchatbot app can be used as SAAS an your clients could use it with their Twilio's account. 

The Midrub Wchatbot app can be limited based on team's role. Each user could decide which team's members will be able to use this app.

About Midrub Wchatbot's Code

The Midrub Wchatbot app has a code adapted for the Midrub CMS and it follows all parameters required for a Midrub app. It has support for Midrub's api, cron jobs, ajax requests, hooks, teams, etc. 

You can translate this app in several languages and all languages directories are located in the language directory. If you want to translate it, please follow the documentation https://www.midrub.com/categories/translation

The hooks located in the directory inc allows you to configure this app in the admin panel, restrict per plan and restrict per team. You can add there even other kind of files.

In the controllers directory you will see 3 files: user, ajax and bot. User is to display the html content to users. Ajax is to process the Ajax Requests. Bot replies to messages in messenger.

In the directory helpers you will find all files created to manage better the code and process only required data. The helpers are based on the controllers directory's files.

In the models directory you will find the app's models which creates the required tables in the database. I like now to use more the Midrub's Base Model for simply SQL queries but for more complex queries i'm using the app's model.

The directory views contains the views files loaded from from the User's controller. 

In this app i've used a new library with font icons which you can find here https://lineicons.com/icons/ This library isn't loaded from app, it is available in the Midrub User's Theme, Luna. I'm providing the theme together with the Midrub Wchatbot app. If you still want to use the Blue Theme, just add in the header from admin the link to icons.

How to configure the Whatsapp numbers for Midrub Wchatbot

Midrub Wchatbot is a Midrub's app which allows to auto reply to the Whatsapp messages. To be able to use Midrub Wchatbot you have to use Twilio and connect Whatsapp numbers approved by Twilio.

To configure the Whatsapp numbers in the Midrub Wchatbot app.

  • Connect the Whatsapp numbers in the Midrub Wchatbot -> Numbers -> Icon with user +.
  • Click on the button Connect Numbers.
  • Enter your Twilio's Account SID.
  • Enter your Twilio's Auth Token.
  • Click on the button Connect.
  • Close the Whatsapp Numbers Manager modal.
  • You will see all connected numbers in the Whatsapp Numbers section.
  • Click on the Connect button at the Whatsapp Number which you want to configure.
  • .You will see the form to add the Default Response. If you want to add one, click on the button Enable too.
  • Then select a category or several categories. The categories are important because only the replies which have the selected categories will reply to the Whatsapp number's messages which have the categories. 


How chatbot replies to messages based on the Whatsapp's messages content

The Midrub Wchatbot app could auto reply to the Whatsapp's messages based on the message content. For this scope, the Midrub Wchatbot app has the section Replies where is possible to create replies and enter the required keywords which should be in Whatsapp's messages to auto reply.

The Replies works in this way:

  • First you have to connect your Whatsapp numbers from the Whatsapp Numbers section.
  • Then, you have to select a category for your Whatsapp numbers.
  • Then, go to the Replies page and click on the New Reply button.
  • Enter the keywords which should be in the Whatsapp messages.
  • Enter the response.
  • Then, in the reply's page you can select the accuracy for your keywords.
  • The most important step is to select the category for the Replies(if you want to auto reply to a Whatsapp Number's messages, the Reply should have at least one category which is selected even for the Whatsapp number).

How to set up a default response for Whatsapp's messages

The Wchatbot app allows you easily to set up a default response which will be sent to users if the bot don't has a reply based on the users messages. 

The default response is easy to create:

  • Go to the Midrub Wchatbot app.
  • Click on the Whatsapp Numbers link.
  • Click on the Connect button at the Whatsapp number(for which you want to set up the default response).
  • In the box Default Response you will see the textarea Text Reply.
  • Enter a default response.
  • Click on Enable.

For default response the categories aren't required. The users will receive the default response if the number has or no selected categories.

Manage the categories in the Midrub Wchatbot app

It's important to know, the categories in Wchatbot could be used in all chatbot apps(Facebook Chatbot, Facebook Marketing, Facebook Commenter, Wmarketing, etc) and you can use the categories created in the Wchatbot for Quick Replies which are sincronyzed with all Midrub's app for chatbots.

The categories are the most important feature because:

  • Them allows to manage easily the Quick Replies. You don't need to take care to asign the Quick Replies every time when you're reconnecting a Whatsapp Number.
  • People which will get the Quick Reply, will be saved with the Quick Replies category and next we can use them for marketing based on the category.
  • Whatsapp Numbers can be easily configured. You just have to select the categories which you've added in some Quick Replies.

The categories can be created and deleted only from the Reply's page. First of all you have to create a Quick Reply to be able to use the Categories.

Get conversations in the Midrub Wchatbot's history

All Chatbot's history can be read in the Midrub Wchatbot's app -> History Page. There you will see the list with the saved conversations. Conversations are saved if the Wchatbot's app has replied based on your keywords parameters or has replied with the default message.

The search input allows to search the conversations based on the Whatsapp number. For example you want to see all conversations of a phone number, you will enter the phone number and the History Page will show you the conversations with paginations if them are more than 10.

In the conversation page, you will see the phone number which has contacted you, time when the bot has received the message, the user's message and the bot's response.

You will see even a user's image which is the default in Midrub(not from Whatsapp).

The conversations can be get even from the reply's page. The reply's page shows in a list all subscribers which have received the response. If you're clicking on the phone number, you will get the conversation.

Get all whatsapp subscribers which have contacted you

One goal of the Midrub Wchatbot app is to collect your potential clients and save them in one or more categories. How this works? First you have to create the Replies, create categories and connect the Whatsapp's numbers. Select categories for the replies and select categories for Whatsapp's numbers. Every time when a potential client will get an auto reply, will be saved in the database with the categories which has the received response.

All people will be saved as subscribers in the Midrub Wchatbot app -> Subscribers Page. There you can find them easily by using the search input or by page. 

In the subscriber's page you will see the phone number of the subscriber, all messages which the subscriber has sent to you and the received response. Below the messages, you will see all categories. You can anytime select or unselect the categories.

The categories of the subscribers will be used in the Midrub Wmarketing app.

How to save and export the phone numbers from the Whatsapp's messages

The Midrub Wchatbot app is able to detect the phone numbers from the Whatsapp's messages and save them in a list. The saved messages can be seen in the Midrub Wchatbot app -> Phone Numbers.

Every time when is saved a new phone number, you will see a notification in the Replies page which is the first page in the Midrub Wchatbot app. With a click you can see the conversation from where was saved the phone number. 

The phone numbers can be managed from the Midrub Wchatbot app -> Phone Numbers. The search input allows to search through the saved phone numbers.

The button Export CSV allows to export all phone numbers in a CSV file.

How to save and export the email addresses from the Whatsapp's messages

Midrub Wchatbot is a Midrub's app which could be used to auto reply to the Whatsapp messages. Moreover, Midrub Wchatbot saved the messages in the database and saves in separate tables the phone numbers and email addresses.

The saved email addresses can be seen in the Midrub Wchatbot app -> Email Addresses. There you will see all saved email addresses. You can search emails by using the search input. The Export CSV button allows to export the saved email addresses in a CSV file.

If you're clicking on an email address you will see the message from where was saved the email address. Also you can bulk delete the email addresses.  

When you have new email addresses you will see the notification in the Replies page.
